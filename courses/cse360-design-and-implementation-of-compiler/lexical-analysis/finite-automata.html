# Nondeterministic Finite Automata

Nondeterministic finite automata (NFA) have no restrictions on the labels of their edges. A symbol can label several edges out of the same state, and ϵ\epsilonϵ, the empty string, is a possible label.

# Transition tables

We can also represent an NFA by a transition table, whose rows correspond to states, and whose columns correspond to the input symbols and ϵ\epsilonϵ. The entry is the transition state corresponding to the current state and the input. If the state-input pair doesn't exist, we put ∅\empty∅ in the entry.

# Deterministic Finite Automata

A deterministic finite automaton (DFA) is a special case of an NFA where:

  1. There are no moves on input ϵ\epsilonϵ.
  2. For each state sss and input symbol aaa, there is exactly one edge out of sss labeled aaa.
